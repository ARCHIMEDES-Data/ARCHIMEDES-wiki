In this section, we’ll introduce the main features that help you navigate the platform quickly and easily.  
  
[ARCHIMEDES](https://www.archimedesdata.ca/) is organized into sections accessible from the top menu. Hovering over a menu item reveals its subsections, and you can return to the Dashboard at any time by clicking ARCHIMEDES home icon. If you ever feel unsure about a feature, you can click the question mark icon to access Help Text for guidance.

We’ll cover the Dashboard, Navigation Path, Help Text, and Data Tables.

### Dashboard 📊

ARCHIMEDES is divided into sections, which are organized in a menu along the top of your screen.
The sub sections details will appear as you hover over any of these menu items. At any point, you
can click on ARCHIMEDES or the home icon to go back to the Dashboard.

💡 **Tip**: If you come across any page that does not display fully, navigate to your browser’s zoom function and zoom in or out. This may be the quick fix to any display issues.

Your dashboard is the first thing you see whenever logging into ARCHIMEDES. What you see on that page will depend on your role and permissions.

### Help Text ❓

Every page in ARCHIMEDES includes a Help panel identified by a ❓ that explains the current section and how to use it.

❓ **Open the Help Panel**  
➡️ Click the question mark icon (?) in the top-right corner of the page. The Help text panel will appear with instructions for that page.

📖 **The Help panel provides**:  
➡️ 🧩 Section overview – a brief explanation of the current section  
➡️ 📝 Detailed instructions – guidance on how to use the page  
➡️ 📋 Step-by-step actions – instructions for completing tasks

❌ **Close the Help Panel**  
➡️  Click Close to hide the Help panel when you are finished.

### Data Tables 📋

Many sections in ARCHIMEDES retrieve information from the database and display it in data tables. These tables include several tools that help you filter, organize, and export your data.

<kbd> <img width="940" height="460" alt="image" src="https://github.com/user-attachments/assets/eec58f97-2687-42ab-9d4c-8458ea2c5cac" /></kbd>

🔎 **Filter Your Data**  
➡️  Use the Selection Filter section to narrow down the data displayed in the table.  
➡️  Enter values in one or more filter fields to refine results.  
➡️  As you type, the system automatically suggests matching data.  
➡️  Click Clear Filters at any time to reset all filters.  

⚠️ : When selecting multiple filter values, the system uses AND logic, not OR.

Example: PSCID: A AND B
This means results must match all selected values.

⚙️ **Advanced Filters**    
Some tables include additional filtering options.  
➡️ Click Show Advanced Filters: Additional filter fields will appear.

These allow you to further refine your search results.

↕️ **Sort Table Columns**
You can easily sort the data displayed in the table.  
➡️ Click any column header to sort by that column.  
➡️ Click again to toggle between:  
     - Ascending order  
     - Descending order  

📄 **Adjust the Number of Rows**  
➡️ If the table contains many results, you can change how many rows appear per page.  
➡️ Use the rows-per-page drop-down menu to increase or decrease the number of displayed rows.  

⬇️ **Download Table Data**  
Most tables include a _Download Table as CSV_ option.  
➡️ Click this button to export the table data as a .csv file, which can be opened in spreadsheet software.
